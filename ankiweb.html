With this add-on when you type "tag:", "deck:", "note:", or "card:" into the browser search bar a dialog opens that allows you to filter and insert a matching tag/deck/notetype name/cardtype name into the search bar. There's also a dialog that puts tags and decknames into one selection dialog. This latter option might be useful if you have many tags and many subdecks. By default this dialog is opened by typing in "all:" or "xx". In the add-on config you can disable the filter dialog for each search tag and set custom strings for the merged deck/tag dialog. There are also shortcuts that open the filter dialogs directly (that you can adjust in the config), e.g. "Ctrl+H, T" opens the tag filer dialog in the browser even if the focus is not in the search box. By default a filter dialog that was opened by a shortcut runs the search. You can adjust this in the config and temporarily override this by pressing shift when you close the filter dialog. In the dialog you can go down with the down arrow or Ctrl+N and go up with the up arrow or Ctrl+H or Ctrl+P. This add-on works with night mode night mode if the night mode version was released on 2020-01-13 or later. # Versions - 2020-01-12 direct shortcuts, night mode compatibility, remembers dialog size - 2020-01-11 initial release # bugs Use this version at your own risk.
